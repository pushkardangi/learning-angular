About

This app is built using the Angular and Bootstrap.Angular is a powerful platform for creating dynamic and responsive web applications. Angular follows the component-based architecture, which promotes modular development and reusability. Bootstrap is Powerful, extensible, and feature-packed frontend toolkit for building UI with HTML, CSS, and JS.

In the app, there's an input field where you can type your task title and description. Press the "Add Todo" button or hit Enter, and the task will be added to your todo list. The list updates dynamically, and you can see your new task immediately.

Absolutely! Each task in the list has a checkbox next to it "Mark as done". Click the checkbox to mark the task as completed. The task's appearance will change to indicate that it's done, helping you keep track of your progress.

You can easily delete a task by clicking the "Delete" button next to the task. This will remove the task from your todo list. Be cautious, as deleted tasks cannot be recovered.

You can contribute to this project by forking this repository from https://github.com/pushkardangi/learning-angular and making a pull request. You can also report bugs and suggest improvements by opening an issue.
